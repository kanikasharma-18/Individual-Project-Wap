# Otaku Oracle

##  Project Overview
Otaku Oracle is a web application that helps users discover what anime to watch next. It fetches real-time data from the Jikan API and allows users to explore trending anime, search by title, filter by genre, and view detailed information.

The name “Otaku Oracle” represents a smart guide for anime lovers, helping them decide what to watch based on their preferences.

This project demonstrates JavaScript concepts like API integration, DOM manipulation, and array higher-order functions.

---

## Features

- Search anime by title  
- Filter anime by genre  
- Sort anime by rating 
- Expandable synopsis section
- Dark mode / light mode toggle  
- Random anime suggestion  
- Debounced search for better performance
